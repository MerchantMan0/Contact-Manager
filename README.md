# About
Basic Contact Manager with the following operations:  
- Registration  
- Login  
- CRUD Operations (Create, Read, Update, Delete)  
-  Search Function  

Stores the following per contact record:  
- Name (Preferred First/Last Name)
- Email
- Phone
- Date Record Created
